Introduction

Introduction

This is the course material for Code Your Future, a not-for-profit volunteer-run coding school for refugees and asylum seekers.

During the course you will learn a technology stack that helps you build anything from a simple website to a powerful database-driven web application or even a mobile app! The six-months course starts with HTML/CSS then core JavaScript, backend development using NodeJS, and modern front-end development with React. We’ll complement the training with workshops on modern software methodologies and job interview practices etc...
